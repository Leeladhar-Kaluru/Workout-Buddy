# Workout Buddy

Workout Buddy is a simple workout tracking app built with the MERN stack.  
It lets users create, view, update, and delete workout entries through a React frontend connected to a Node.js and Express backend with MongoDB Atlas.

## Live Demo

- **Frontend:** https://workout-buddy-jade.vercel.app/login
- **Backend API:** https://workout-buddy-1imk.onrender.com

## Features

- Create workout entries
- View all workouts
- Update workout details
- Delete workouts
- REST API integration
- Responsive frontend
- MongoDB Atlas database
- Deployed frontend and backend
